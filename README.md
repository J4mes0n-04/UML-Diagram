# Примеры работ с диаграммами BPMN, UML, демонстрация навыков моделирования бизнес-процессов и проектирования архитектуры ПО.

##  Инструменты, которые использовались:

- Mermaid
- Draw.io (набор BPMN 2.0)
- Miro
- и др.

## Как пользоваться репозиторием?
Ознакомьтесь с разделами по интересам — BPMN или UML.
Изучайте примеры по тематике, которая вас интересует (например, «документооборот» или «микросервисы»).
Используйте код PlantUML/BPMN.io для своих проектов — все примеры открыты для изучения.
Смотрите комментарии к файлам — там указаны инструменты, версии и ключевые особенности каждой диаграммы.
